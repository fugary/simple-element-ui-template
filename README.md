# simple-element-ui-template
简单的基于VueJS和ElementUI的后台页面，最基本的整合布局，暂没有很多附加功能。

### 基于框架

1. Vue + Vuex + Vue-Router
2. ElementUI
3. vuex-persistedstate
4. axios
5. moment

目前并没有做太多集成，主要基本布局、Http请求、主题切换等。
